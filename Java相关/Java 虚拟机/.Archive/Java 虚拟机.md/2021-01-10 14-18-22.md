# **Java 虚拟机**
## Java 虚拟机的执行流程
Java 虚拟机的执行流程分为两部分：编译时环境和运行时环境。当一个 Java 文件被 Java 编译器编译成 Class 文件后，才能由 Java 虚拟机进行处理。Java 虚拟机与 Java 语言没有必然的联系，它只与特定的二进制文件：Class 文件有关。

## 类的生命周期
1、**加载**：查找并加载 Class 文件
2、**链接**：包括**验证**、**准备**和**解析**
* **验证**：确保被导入类型的正确性
* **准备**：为类的静态字段分配字段，并用默认值初始化这些字段
